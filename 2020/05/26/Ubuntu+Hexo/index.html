    Ubuntu18.04下搭建Hexo
    发表于 2020-05-26| 更新于 2020-05-26|-教程
    阅读量:

    写在前面的话:

    本博客针对Ubuntu18.04下Hexo部署进行总结，希望对你有帮助。
    欢迎交流（email：1824425412qq.com)
    本文参考：hexo文档

    目录

    1. 安装Nodejs，Npm  
2. 安装Hexo  
3. 安装Git  
4. Hexo初始化  
5. 部署在Github上  
6. 主题，自定义域名等

    开始前先进行ubuntu软件库更新 

    sudo apt-get update
sudo apt-get install -y software-properties-common
sudo add-apt-repository ppa:chris-lea/node.js
sudo apt-get update

    1. 安装Nodejs,Npm

    sudo apt-get install nodejs  
sudo apt install npm

    2. 安装Hexo

    sudo npm install hexo-cli -g

    3. 安装Git

    sudo apt-get install git

    生成公钥秘钥 

    git config --global user.name "yourname"
git config --global user.email "youremail"
ssh-keygen -t rsa -C "youremail"

    在根目录下 

    ls -a

    然后打开.ssh文件下的id_rsa.pub文件，这是公钥，复制他然后打开你的github主页：头像->setting->SSH and GPG keys->New SSH key,保存。
    然后在终端输入：

    ssh -T git@github.com

    成功显示Hi yourusername! You’ve successfully authenticated, but GitHub does not provide shell access。

    4. Hexo初始化

    建立一个文件夹，用于存放你的Hexo本地文件，cd 到这个目录中，初始化hexo: 

    hexo init
npm install //安装插件
hexo g //生成静态文件
hexo s//本地浏览

    这时候就可以用浏览器打开网址： http://localhost:4000/ 来进行预览了。Ctrl+C退出。

    5.部署在github上

    (1)repo建立

    在github中新建一个存储库（默认设置就好），库名格式必须为：your_github_username.github.io(例如：我的github用户名为luaogithub，那么我建立的存储库名称为：luaogithub.github.io)

    (2) 配置hexo博客根目录下的_config.yml文件（！！注意每个冒号后都有一个空格）

    # Deployment
## Docs: http://hexo.io/docs/deployment.html
deploy:
type: git
repo: git@github.com:/your\_github\_username/your\_github\_username.github.io.git
branch: master

    (3)本地进行hexo部署

    hexo中的git插件需要手动安装之后才能进行hexo在git上的部署： 

    npm install hexo-deployer-git --save

    然后执行

    hexo clean  //清除缓存文件 (db.json) 和已生成的静态文件 (public)。
hexo g  
hexo d  //部署到github上

    之后需要等待几分钟，打开浏览器，复制你的github仓库名，即可作为网址打卡你的个人博客了。出现404或者无法访问时，可以检查相关配置是否正确，只要细心就一定可以部署好。

    (5)主题，自定义域名

    笔者的个人博客主题为hexo-theme-butterfly，有需要可以移步去进行更换，教程写得很详细。

    关于自定义域名，可以参考域名绑定这篇博客。这里不做介绍。

    以上就是关于Ubuntu环境下搭建Hexo博客的介绍，如有错误希望可以指出和包含，谢谢。觉得写得好的也可以在底部点击答赏，请笔者喝杯咖啡。
